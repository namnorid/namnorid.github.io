SOP: Standard Operating Procedure
Start

Begin by collecting the equipment listed in 5S. After the pre-production meeting, go to the pallet square and start the browser. Go to lmk.posten.no use the login marnikub. See current password at the main office. Enter the menu → select workstation → select ‘pallstation’. Go to 'In to hub / terminal'. Select the input field and scan the barcode on the label.

Ordinary work

On the pallet square, product 0336, ‘styckegods’ (freight goods), HD volumes and load carriers are placed.

Begin scanning the pallets starting at gate 44 and go down the pallet square; scan the pallets in a row. When scanning the pallets, add extra service codes (VAS) in these cases:

•  0073 Oversize (0336): If the pallet size exceeds 120cm in length, 80cm in width or 180cm in height.

•  0073 Oversize (0336+0062): If the half pallet exceeds the dimensions 60cm in length, 80cm in width or 150cm in height.

•  0073 Oversize (0336+0061): If the quarter pallet exceeds the dimensions 30cm in length, 40cm in width or 120cm in height.

•  0067 Missing EDI: If the barcode lacks all address information.

•  0060 Correction of delivery address: If the address or postal code is incorrect and needs to be corrected.

•  1070 Feilkodet: If the customer has the wrong product code.

•  1132 Feil i EDI: You use this for other errors made by the customer in EDI, for example the wrong service code.

•  1153 Barcode not readable: If the bar code cannot be read with a scanner, for example bad barcode.

•  1154 Incorrectly placed barcode: If the barcode is placed so that it cannot be scanned, for example over an edge.

•  1156 Barcode hidden by plastic: If the barcode is covered with plastic or tape.

When the pallet is scanned and deviations are entered, press save. The LMK window states which DIP number the pallet is going to. Write the DIP number on a white label and place it on the pallet well visible to the pallet drivers. Write the numbers so they are well readable.

Try to have short and effective conversations with drivers and other staff so that we spend as much as possible on scanning the pallets. Use of a mobile phone for private purposes may only be done if there is time and with minimal impact on production.

End of production

Once all trucks have departed, go into the office and check which cleaning area you have been assigned. Clean the area according to the cleaning instructions. Do not forget to return the equipment used during the night (Walkie-talkies, forklift, etc.). Ask the cleaning supervisor where help is needed.
