Responsible: Carl Johansson
Assignment: Cargo shipment handling
Section: HUB - Production - Pallets
Process: Handling of cargo shipments
Updated: 2019-09-29
SOP: Standard Operating Procedure
Start

Cargo goods are handled in LM. For LM operating instructions, see SOP for Pallet Handlers. All goods must be scanned the same day it arrives for it to be correct in invoicing.

Ordinary work

Additional codes (see fig. 1):

  •  Most are added upon booking or automatically added based on the dimensions entered.

  •  LQ & ADR are the only additions we need to add manually if they are missing.

  •  Other additions appear as "free text" for Goods Sweden but do not change the invoice. Can be used for informational purposes, such as "new label printed."

Customer-specific:

  •  Profsafe – Should not be corrected. Do not change any dimensions in LM except if dimensions are completely missing; Then insert the correct one.

  •  Rexab – Has an agreed maximum height of 2.15m.

Dimensions and volume:

  •  If the goods are not stackable specify height 141cm in LM.

  •  Goods with height above 2.10m are stopped and placed on the UFO surface in the northern hall.

  •  Shipments of 4 pallets or more are counted as batch and placed on the batch surface in the northern hall. Also applies if the total weight exceeds 2499 kg.

EDI:

  •  If EDI is missing, we must not add it; but the shipment should be placed on the EFO surface.

  •  If part of EDI is missing or incorrect, eg postal code, it can be adjusted immediately and the shipment forwarded.

Remaining goods:

  •  Scanned as remaining goods, like Parcels and HD.

Other:

  •  All goods must be scanned the same day as they arrive in order for them to be correct in invoicing.

Deviation report

  •  Batch shipments – Specifies the shipments to the same receiver that have arrived during the production day.

  •  Deviations / UFO – Specifies goods where action is required before it can forwarded.
