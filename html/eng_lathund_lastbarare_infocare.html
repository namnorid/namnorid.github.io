Create load carrier for Infocare SOP

SOP: Standard Operating Procedure

The following description applies to the creation of load carriers for Infocare in LM.

First click on Logical Carrier (in the left sidebar, under Carrier.)

Click in the box for Create Logical Carrier.

Choose Jönköping hub in the dropdown for Terminal, and JonkopL2 for Select printer.

Choose Infocare … in the dropdown for Destination.

Finally, press Save.
