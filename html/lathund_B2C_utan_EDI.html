SOP: Standard, Operating, Procedure

Quick guide

When we receive packages with product code 0340 or 0342 without EDI and we manually register these packages, we must ensure that, in addition to address, customer number, product, we make sure to register the following:

  -  The customer's mobile number or email address so we can notify the customer.

  •  Add if written on the label.

  -  If the customer has chosen, 0091 eAdvising should be added.

  -  Always add VAS 1167 (letter notification). This is so that letter notification to recipients will expire when the parcel has reached a representative if nothing else works.
