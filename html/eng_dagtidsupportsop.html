Daytime support SOP

Responsible: Sofie Wahlström
Assignment: Unloading daytime
Section: HUB – Production – Pallets
Process: Unloading daytime
Updated: 2019-09-27
SOP: Standard Operating Procedure
Start

Begin by collecting the equipment specified in the 5S.

Ordinary tasks

The main task is to unload arriving cars, scan pallets (product code 0336), and drive scanned pallets to the respective area.

1.Inform the supervisor that you have arrived and get a brief description of what is needed.

•For current cars see the Traffic Report (F:\Bring\Parcels\PR Production\PR Jönköping\PR Hub\Nattrapport\Trafikrapport 2019..)

•When unloading, count the car's total amount of pallets and inform the supervisor.

•Always help with unloading the arriving cars through the day.

2.At 13:00 help sorting the 24-hour customers for Finland. See that SOP for more info.

3.After the Finland sorting is finished, unload STO-JKP 1 and DKE-SEJ 2.

4.Goods from Kicks

a.See mail from the traffic manager: "Ikväll tidig" for the number of arriving Kicks cars, and other extra cars.

b.Scan unloaded goods (product code 0336) from the cars from Kicks. See the guide for more info.

c.Always start by checking whether the pallets have got EDI. This is done by checking the outermost pallets on the rig.

d.If the goods are LQ, i.e. limited amount of dangerous goods, check that additional code 0003 is specified. If the add-on code is missing, add it.

e.When all pallets are registered in LM, help with driving out the pallets.

5.When all Kicks goods are scanned, scan pallets on the short side of the terminal. If there is nothing to scan, help with unloading the other cars / driving pallets to their respective gate / pallet area.

6.When the evening staff arrives, you will receive information from the evening's team leader about which position to do.

Notices

•If there is nothing to do, talk to the supervisor about additional tasks. Helping with driving pallets from the Return Dep. and the approved goods from the Customs area, is needed daily. Both located in the Northern hall.

End of production

Clean the forklift and your workstation after the work shift is complete.
